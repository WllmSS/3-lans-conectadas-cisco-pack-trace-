Comunicação entre 3 LANs – Cisco Packet Tracer

Objetivo

Implementar uma rede composta por três LANs independentes interligadas por um roteador, permitindo comunicação entre todos os dispositivos através de roteamento IPv4.

Topologia
- 1 Roteador Cisco 2811/2911
- 3 Switches
- 9 PCs (3 por LAN)
- 3 Impressoras (1 por LAN)
- Endereçamento
- Rede	Faixa	Gateway
- LAN A	192.168.10.0/24	192.168.10.254
- LAN B	192.168.20.0/24	192.168.20.254
- LAN C	192.168.30.0/24	192.168.30.254
- Funcionalidades Implementadas
- Configuração de três sub-redes independentes.
- Endereçamento IPv4 estático.
- Configuração de gateways padrão.
- Roteamento entre redes utilizando um roteador Cisco.
- Compartilhamento de impressoras entre as LANs.
- Testes de conectividade utilizando ICMP (Ping).

Tecnologias
- Cisco Packet Tracer
- IPv4
- Switching (Camada 2)
- Routing (Camada 3)
- TCP/IP

Resultados

Todos os dispositivos conseguem se comunicar entre as três redes, validando o funcionamento do roteamento e da infraestrutura proposta.
